# 📚 BookShelf - Smart Library Platform

A modern AI-inspired university library system built using React.

## 🚀 Features
- 🔍 Smart search system
- 📖 Book recommendation UI
- 🏷️ Tag-based feedback (easy, medium, advanced)
- 💾 Save/bookmark books
- 👤 Login & Signup (basic backend)

## 🛠️ Tech Stack
- React.js
- CSS
- Node.js (basic backend)
